# 🏦 Sistema Bancário em C

Sistema bancário desenvolvido em linguagem C com foco na prática de conceitos fundamentais e na construção de uma aplicação completa do zero.

## 🚀 Funcionalidades

* Criação de contas bancárias
* Login de usuário (CPF e senha)
* Depósito e saque
* Transferência entre contas
* Validação de dados (nome, CPF, senha)
* Separação entre modo administrador e usuário

## 🧠 Conceitos Aplicados

* Ponteiros
* Structs
* Modularização (.c e .h)
* Manipulação de strings
* Busca linear
* Validação de entrada de dados

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de consolidar conhecimentos em C, simulando um sistema real e organizando o código de forma clara e escalável.

## 📌 Melhorias Futuras

* Persistência de dados em arquivos
* Implementação de extrato bancário
* Sistema de autenticação mais robusto
